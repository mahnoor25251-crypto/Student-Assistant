# Student-Assistant
student-assistant-system

A Python-based Student assistant system built with core Python fundamentals.

## Features
- Bulk student registration with validation
- Department-wise filtering (CS, Data Science, Cyber Security, IT)
- Timetable & marks management
- Auto grade calculation (A+, A, B, C, Fail)
- Export records to file

## Tech Stack
Python 3 | File Handling | Dictionaries | Functions
